# GoClaw Blog

Static HTML blog for the [GoClaw](https://goclaw.sh) project. Deployed on Vercel.

## Adding a new post

1. Create `public/<slug>.html`
2. Add entry to `index.html`
3. Push — Vercel auto-deploys

## Local preview

```bash
npx serve .
```

Visit `http://localhost:3000/posts/<slug>` to test rewrites locally (or open HTML files directly).

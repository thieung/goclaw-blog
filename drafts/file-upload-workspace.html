03 — Security Model

Tenant isolation & path safety

Every upload is scoped to a tenant+team directory. Four security checks run before any byte is written to disk.

Symlink Escape Prevention
EvalSymlinks resolves the real path and verifies it stays within the tenant scope directory before writing.
Blocked Extensions
Executable types (.exe, .sh, .bat, .cmd, .ps1, .com, .msi, .scr) are rejected both client-side and server-side.
File Quota
100-file limit per scope directory (MaxFilesPerScope). Count checked before write — no silent quota overflow.
Tenant Isolation
Path scoped to data/{tenantId}/{slug}/teams/{teamId}/ — cross-tenant access structurally impossible.

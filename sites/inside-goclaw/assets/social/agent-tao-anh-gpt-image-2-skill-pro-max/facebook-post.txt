🦊 Ae nào đang trả ChatGPT Pro hằng tháng mà agent vẫn phải xài API key riêng để gen ảnh thì cái này đáng đọc: GoClaw v3.11.0 mở native path qua Codex OAuth, quota gpt-image-2 trong subscription xài được luôn, không phát sinh API Usage cost thêm.

Vấn đề cũ: tool create_image cần APIKey + APIBase chuẩn OpenAI, còn ChatGPT OAuth thì dùng refresh token và endpoint chatgpt.com/backend-api, hai cái không khớp nhau. Nên quota nằm yên trong tài khoản Pro, agent không gọi được.

PR #1002 giải quyết 3 điểm:

🔹 Wire Codex Responses API làm path riêng cho image gen, token tự refresh theo lifecycle OAuth, không cần quản lý key thủ công
🔹 Gating 2 tier: model phải khai báo capability "image_output" + agent phải bật cờ allow_image_generation (mặc định bật, dễ kiểm soát)
🔹 PNG provenance: embed prompt + revised_prompt + agent name vào tEXt chunk của file PNG, 6 tháng sau mở lên vẫn biết ảnh sinh từ context nào

Riêng phần prompt thì có thêm skill gpt-image-2-pro-max do Richard Ng (contributor GoClaw) open-source, license MIT. Skill được package theo chuẩn Claude Code Skill convention và tương thích Skill system của GoClaw, tức là zip thư mục skill lên upload qua Web Dashboard là agent dùng được ngay, không phải cài gì thêm.

Bên trong skill có corpus 3,000+ prompt template từ creators Twitter/X. Skill index lại, dùng BM25 search, mỗi kết quả trả về đều cite tác giả gốc. Upstream credit cho repo EvoLinkAI/awesome-gpt-image-2-prompts, nhưng bản index + search engine là của Richard build thêm.

Ghép lại: agent search corpus → pick template phù hợp mood → refactor prompt → gọi create_image qua Codex OAuth → PNG ra kèm provenance. Tiền chỉ tốn cái subscription đã trả rồi 🔥
